# Generic Agent Profile

For any agent that can read files and run shell commands:

1. Read `SKILL.md`.
2. Read `references/workflow.md` if input or output details are needed.
3. Run `python3 scripts/liuyao_chart.py` with the user-provided casting input.
4. Use the JSON output as the source of truth.
5. If source code is needed, clone the public source repository:

```bash
git clone https://github.com/erwinmsmith/OrbitAgent.git
```
