# Few-Shot Patterns

These examples demonstrate style and grounding. Do not copy their chart facts into a different reading.

## Example 1: Structured Project Reading

User:

```text
这个项目上线顺不顺利？
```

Chart facts from the bundled script:

```text
本卦 火水未济，变卦 天水讼，动爻第5爻。
问题类型 项目产品。
主要用神 子孙在第2爻和第5爻；第5爻子孙未土发动，旬空，旺衰标签为相、得月生、得日生、旬空，化申金妻财，relation 为化生。
世爻第3爻兄弟午火，旬空；应爻第6爻兄弟巳火，旺。
第5爻未与日辰午相合。
伏神：第3爻官鬼亥水伏于兄弟午火，伏克飞。
```

Assistant style:

```markdown
## 结论先行

从这卦看，项目不是完全卡死，但上线过程更像"先补尾工、再处理争议"：有产出推进的信号，也有空亡和变卦讼象带来的沟通/验收阻力。更适合按阶段推进，先把可交付物、验收边界和责任分工定清。

## 排盘摘要

本卦为火水未济，变卦为天水讼，动爻在第5爻。问项目产品，主要看子孙；子孙在第2爻和第5爻，其中第5爻发动，化出妻财。

## 用神与世应

子孙主产品和产出。第5爻子孙未土发动，并得月日生扶，说明事情有推进和产出基础；但该爻又逢旬空，表示当前成果、资源或承诺还没有完全落地。世爻兄弟午火也空，代表自己这一侧的状态还不够稳，容易有计划和执行之间的落差。

## 动爻与变卦

第5爻子孙动而化妻财，且动化关系为化生，说明产品产出有转向商业结果或收益目标的可能。问题在于变卦为讼，后续更容易出现标准、权益、验收、沟通上的争执，因此顺利与否不只看开发完成，还看上线前的确认机制。

## 综合判断

整体偏向"能推进，但不会特别省心"。有利点是用神发动并得日月生扶；风险点是用神与世爻皆有空象，且变卦见讼。现实上应优先补齐上线清单、验收口径、负责人和回滚方案，避免临门一脚因沟通问题拖延。
```

## Example 2: Missing Context

User:

```text
看看这个
```

Assistant style:

```markdown
我可以解，但现在问题太短，无法判断该取什么用神。请补一句你要问的具体事项，例如事业、感情、项目、求财、健康、失物或某个决定；如果已经有卦，也把六个爻值或排盘结果发来。
```

## Example 3: Health Boundary

User:

```text
这卦看我的病会不会严重？
```

Assistant style:

```markdown
这类问题可以从卦盘里看压力点和需要关注的方向，但不能替代医生诊断。请先说明症状、就医情况和你真正想问的是"是否需要复查"、"恢复趋势"还是"治疗选择"；如果有急性疼痛、呼吸困难、出血、意识异常等情况，应立即就医。

等排盘完成后，我会只基于卦盘中的官鬼、子孙、世爻、日月旺衰、空破和动爻来做谨慎分析，不下确定性医疗结论。
```
